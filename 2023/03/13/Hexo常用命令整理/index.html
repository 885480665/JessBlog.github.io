Hexo常用命令整理

发表于2023-03-13|更新于2023-03-15
|字数总计:171|阅读时长:1分钟|阅读量:

前言:

​ 为了方便查阅,这里将常用指令汇总。

简写命令：

缩写 全称
hexo i hexo init
hexo n “我的第一篇文章” hexo new “我的第一篇文章”
hexo new post “我的第一篇文章”
hexo p hexo publish
hexo g hexo generate
hexo s hexo server
hexo c hexo clean
hexo d hexo deploy
hexo deploy -g hexo deploy –generate
hexo generate -d hexo generate –deploy

命令详解：

hexo init

hexo init 命令用于初始化本地文件夹为网站的根目录

folder 可选参数，用以指定初始化目录的路径，若无指定则默认为当前目录。

hexo new

hexo new 新建文章，一般可以简写为 hexo n
